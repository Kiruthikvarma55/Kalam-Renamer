# Kalam-Renamer

An Open Source Telegram Renamer RoBot

**My Feature**:

You can Rename any Telegram Files with Custom Thumbnail Support


### You can also tap the Deploy To Heroku button below to deploy straight to Heroku!

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://www.heroku.com/deploy?template=https://github.com/kiruthikvarma55/Kalam-Renamer)


#### The Hard Way

```sh
virtualenv -p python3 VENV
. ./VENV/bin/activate
pip install -r requirements.txt
cp sample_config.py config.py
--- EDIT config.py values appropriately ---
python bot.py
```
